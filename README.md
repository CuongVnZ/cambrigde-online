## Setup
There are two main code: Client and Server (Frontend & Backend) 
using npm or yarn

### Server
1. Install dependencies
``` bash
yarn install
```
2. Start server
``` bash
yarn start
```

### Client

1. Install dependencies
``` bash
yarn install
```
2. Start client
``` bash
yarn start
```
